# LOGO EDITOR

***

## Installation

```shell
cd xxx
```

其中xxx为$Makefile$所在目录

```shell
make
```

系统会生成编译$interpreter$/$interpreter.exe$

```shell
python3 logo.py
或
python logo.py
```

***

## Requirement

```shell
g++(需支持C++17)
python 3.5+
	opencv-python(可以使用pip install opencv-python获取)
	tkinter(python3内置)
```
